About Underscore KB

Underscore KB is a knowledgebase for beginning developers to better understand the syntaxes they may come into contact with. The purpose of this is to demonstrate segments of code so that people who are learning have an easily accessible reference to better understand the tools they are deploying.

I've started this project as a learning tool for myself so I can better understand the tools I need to attain the career I want. Feel free to use this tool for your own learning. Also, please feel free to leave any feedback on my design or implementation. Please keep in mind that, as of this writing, I do not know Redux, React, or many other JS Frameworks that make this process easier. I am also not experienced with responsiveness, so I will work on that as I go, as well. Currently, this website is best viewed on a computer at 1920x1080 resolution. I am aware of responsiveness issues. Please bear with me.

You can click the Next Card button to swap to a new card.

Using this Website

The layout for this website will initially be exactly what you see here. A user will choose a language and be able to click through cards that teaches them various implementations of syntax such as centering text or manipulating shadows, etc.

Eventually, I want to add a Wiki Mode button that will swap the entire layout for this website into a wiki format.

Please choose a language from the nav bar at the top of the page to begin exploring!
